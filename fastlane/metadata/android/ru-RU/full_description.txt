EtchDroid — это свободное приложение с лизензией GPLv3+, помогающее записывать образы на USB-накопители.
Используйте его для создания загрузочного USB-накопителя GNU/Linux, если ваш ноутбук помер или вы находитесь в глуши.

Особенности:
• Поддерживает сырые образы дисков и ISO-образы большинства дистрибутивов GNU/Linux.
• Поддерживает образы Raspberry Pi (сначала распакуйте ZIP-файл; используйте SD-USB адаптер, внутренний слот SD не поддерживается).
• Поддерживает большинство DMG-образов (бета)
• Без рута, всё делается в рамках Android API
• Free as in freedom

Планируется:
• Запись распространенных дистрибутивов GNU/Linux в потоковом режиме из Интернета без хранения на устройстве
• Поддержка ISO для установки Windows (скорее всего, потребуется рут)
• Поддержка прошивки образа в стиле Unetbootin

Исходный код размещён на GitHub: https://github.com/Depau/EtchDroid

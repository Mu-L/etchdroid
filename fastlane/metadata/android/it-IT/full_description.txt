EtchDroid è un'app open-source che consente di scrivere le immagini ISO su chiavette USB.
Usalo per creare una chiavetta di installazione del sistema operativo quando il tuo computer non parte più.

⭐️ Dispositivi supportati ⭐️

✅ Chiavette USB
✅ Adattatori SD card USB
❌ Hard disk o SSD USB
❌ Dock e hub USB
❌ Slot interno SD card
❌ Lettori di dischi ottici e floppy
❌ Dispositivi Thunderbolt


⭐️ Tipi di immagini supportate ⭐️

✅ Immagini di sistemi operativi GNU/Linux moderni, inclusi (ma non solo) Arch Linux, Ubuntu, Debian, Fedora, pop!_OS, Linux Mint, FreeBSD, BlissOS
✅ Immagini SD card per Raspberry Pi (a patto che le estrai prima dall'archivio!)
❌ ISO di Microsoft Windows ufficiali
⚠️ Immagini di Windows create dalla comunità specificamente per EtchDroid (attenzione: potrebbero contenere virus!)
❌ Immagini disco Apple DMG
❌ Immagini di sistemi operativi GNU/Linux vecchi < 2010, ad esempio Damn Small Linux


Il codice sorgente è disponibile su GitHub: https://github.com/EtchDroid/EtchDroid
